## ⚙️ How to Use the Interface

The interface centers on the Spectrogram displays. Time sits on the x-axis, and frequency sits on the y-axis. The intensity of color represents the volume level of a specific frequency. 

Select the tool that matches your task. Use the Lasso tool to draw around a shape in the spectrogram, or use the Brush tool to highlight specific frequency bands. After you make a selection, you apply an effect or move the audio data to a new layer. This keeps your original file intact while you test different edits.
